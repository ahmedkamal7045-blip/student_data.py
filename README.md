Student Data Management System (CSV)

Project Overview
    This is a Python project for managing student data using CSV files as storage.

Features
- Add and store student data  
- Read and display student records  
- Update existing data  
- Search for a student by name or ID  
- Delete student records  
- Organize structured data using CSV files  

Technologies Used
- Python  
- CSV file handling  

What I Learned
- Working with CSV files in Python  
- Reading and writing structured data  
- Implementing search and delete operations  
- Data organization and basic data management  
- Problem solving and logical thinking  

How to Run
python student_data.py
